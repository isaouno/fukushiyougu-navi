# 福祉用具おすすめナビ

身長、体重、歩行能力、利用場所、住環境、トイレ・排泄状況などを入力すると、福祉用具の候補をカテゴリ別に3件ずつ提案する静的Webアプリです。

## 公開用ファイル

GitHub Pagesなどで公開する場合は、`docs` フォルダを公開元にしてください。

- `docs/index.html`
- `docs/styles.css`
- `docs/app.js`
- `docs/.nojekyll`

## ローカルで開く

ブラウザで確認する場合は、`open_in_browser.bat` をダブルクリックしてください。Edge、Chrome、既定ブラウザの順で `index.html` を開きます。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダの内容をリポジトリにアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. `Branch` を `main`、フォルダを `/docs` にして保存します。
6. 数分後に表示されるURLを共有します。

## 注意

このアプリは福祉用具選定の候補出しを目的とした参考ツールです。実際の選定・利用可否は、利用者の身体状況、住環境、介助体制を確認したうえで、福祉用具専門相談員、ケアマネジャー、医療・リハビリ職などの専門職が判断してください。

カタログ由来の商品名・仕様・掲載ページなどを外部公開する場合は、提供元カタログの利用条件や許諾範囲を確認してください。
