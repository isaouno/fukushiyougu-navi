# サイトに貼り付けて公開する方法

## 推奨: iframeで貼り付ける

一番安定する方法です。既存サイトのデザインに影響しにくく、アプリ側のCSSやJavaScriptも動きやすいです。

1. `fukushiyougu_app.html` を公開できる場所にアップロードします。
   - 自社サイトのファイル置き場
   - GitHub Pages
   - Netlify
   - サーバーの公開フォルダ
2. アップロード後のURLを取得します。
3. `embed_iframe_code.txt` の `src` をそのURLに差し替えます。
4. 差し替えたiframeコードを、サイトのHTML埋め込み欄に貼り付けます。

## 直接貼り付ける場合

サイト側が `<style>` と `<script>` の貼り付けを許可している場合は、`fukushiyougu_app.html` の中身をそのまま貼り付けても動きます。

ただし、WordPress、Wix、Jimdo、Studioなどでは、プランや設定によって `<script>` が削除されたり無効化されたりすることがあります。その場合はiframe方式を使ってください。
